1) Démarrage rapide

Choisir un dossier
  • Chrome/Edge recommandé. Certains dossiers système Windows sont refusés.
  • Tu dois avoir l’écriture dans ce dossier.
Choisir un mode
  • KPI factu Attachements Prod 8 LOT Tout le dossier (libre)
  • Tu peux enregistrer tes propres modes.
Aperçu puis Renommer
  • Aperçu montre exactement les nouveaux noms.
  • Renommer applique. Undo disponible.

2) Presets conseillés

Chaque preset remplit mode, patron, extensions, heure, mapping.

  • KPI AURA — _KPI factu AURA {DATE:dd.MM} (.csv)
  • Attachements 6 régions — export_attachements{LABEL}{DATE:ddMM} (.csv) + mapping 1..6
  • Prod 8 régions 12h — Avancement prod {REGION} {DATE:dd.MM} {HOUR}h (.xlsx, H=12)
  • Prod 63/74 décalage — Avancement prod {NUM} {DATE:dd.MM} {HOUR}h (détecte 63/74)
  • LOT FLASH Semaine — LOT {LOT} {REGION} S{WEEK} (.xlsm)
  • Créer 6 régions — Avancement prod {NAME} {DATE:dd.MM} + liste AURA…PRM

Dans l’appli, clique les boutons “Presets 1-clic”.

3) Jetons rapides

{DATE:dd.MM} {DATE:ddMM} {WEEK} {INDEX} {INDEX:3} {LABEL} {REGION} {HOUR} {LOT} {NUM} {NAME} {EXT}

Le mapping Index→Libellé nourrit {LABEL}. En LOT, si un numéro de LOT est détecté, il est prioritaire.

4) Renommer

  1. Sélectionne le dossier.
  2. Choisis le mode (dont “Tout le dossier (libre)”).
  3. Règle Heure (auto/12 h), Date, Extensions.
  4. Complète la table Index→Libellé si tu utilises {LABEL}.
  5. Clique Aperçu → vérifie les colonnes.
  6. Clique Renommer. Conflits → suffixes “-1, -2…”.
  7. Besoin de revenir ? Annuler (Undo).

Prod 8 : l’heure est retirée pour CVL/DOE/IDF/PRM. Attachements : ordre par mapping si défini, sinon AURA→PRM.

5) Créer des fichiers

Avec liste

Patron : Avancement prod {NAME} {DATE:dd.MM}

AURA Corse CVL DOE IDF PRM
Sans liste

Patron : Lot {INDEX} fichier, Quantité=3

Résultat : Lot 1/2/3 fichier

Sécurité
  • ✅ prêt si libre
  • ⚠️ existe déjà si conflit
  • ❌ nom vide si patron ne produit rien

L’aperçu création n’écrit rien. “Créer” dépose des fichiers vides.

6) Dépannage

  • Je vois juste “.csv/.xlsm”. → ton patron ne produit rien (ex. {LABEL} vide). Remplis la table, change Index départ, ou ajoute {NAME}/{INDEX}.
  • Impossible d’ouvrir le dossier. → Choisis un dossier utilisateur (Bureau/Documents), pas un dossier système.
  • Mauvaise heure. → force l’heure via “Heure cible (24 h)”.
  • Conflits de noms. → suffixes automatiques “-1”, “-2”…
  • Mes modes persos ont disparu. → localStorage du navigateur ; si tu changes de profil/PC, il faut les recréer.

7) Tips pro

  • Enregistre tes combinaisons en modes personnalisés (nom lisible).
  • Les presets 1-clic te font gagner 80% du temps ; ajuste seulement l’exception.
  • Pour les attachements, la table Index→Libellé impose ton ordre (remplace AURA→PRM).
  • En LOT, mapping par numéro de LOT en priorité (si détecté).
